# 🔋 Smart EV Battery Protection System

A multi-sensor battery monitoring and protection system using **ESP32**. It monitors **Voltage, Current, Temperature, and Gas** to detect battery faults and automatically disconnect the load using a relay.

## ✨ Features
- Real-time battery monitoring
- Voltage, Current, Temperature & Gas sensing
- dV/dt, dT/dt & Power calculation
- Pre-fault and fault detection
- Automatic relay protection
- Dataset generation for TinyML/AI

## 🛠 Hardware
- ESP32 DevKit
- INA219 Current Sensor
- DS18B20 Temperature Sensor
- MQ-2 Gas Sensor
- Voltage Sensor
- TP4056 Charger
- XL6009 Boost Converter
- 2-Channel Relay
- 18650 Li-ion Battery
